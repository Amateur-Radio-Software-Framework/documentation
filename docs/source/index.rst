Welcome to Amateur Radio Software Framework Documentation!
==========================================================

Here you will find all the documentation for the Amateur Radio Software Framework. Our goal is to establish some best practices around developing software for amateur radio. Please feel free to navigate around and see what we have available. Please also note this documentation will be a living document and may be updated at any time. You are also welcomed to contribute to this documentation if you have anything you would like to add. The source code for the documentation can be found on `GitHub <https://github.com/Amateur-Radio-Software-Framework/documentation>`_.
